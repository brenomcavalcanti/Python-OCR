# Python-OCR

Um leitor de tela (OCR) usando Pytesseract em Python para exclusivamente ler múltiplos números decimais, com threading para multitarefas e parte de uma RPA mais complexa (não deve ser dificil adaptar para a leitura de letras ou outras de suas necessidades).

Conta com uma forma de salvar e criar backups dos dados adquiridos, verificação do item coletado por repetição e comparação e uma função de auto-report de itens que não deveriam entrar na lista, para garantir sua pureza e identificar problemas, além disso, salva o item coletado, a localização que ele ocuparia na lista e uma imagem do item antes de ser iterado.
